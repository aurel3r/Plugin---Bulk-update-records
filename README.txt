	Установка:

Прежде всего, вам нужно загрузить файл плагина и распаковать его (если вы читаете это, вы, вероятно, уже сделали это).
Теперь вам нужно выполнить одно из следующих действий:

1.Вариант.В панели администрирования WordPress откройте страницу Плагины -> Добавить новый -> Загрузить.
Выберите и загрузите woocommerce-advanced-bulk-edit.zip файл и нажмите Активировать плагин.
2.Вариант.Распакуйте woocommerce-advanced-bulk-edit.zip архив в папку плагинов WordPress (находится по адресу /wp-content/plugins/).
Откройте свой браузер и перейдите в панель администратора WordPress.
Откройте страницу Плагины -> Установленные плагины и активируйте плагин WooCommerce Advanced Bulk Edit.

	Как использовать:
-Плагин доступен через опцию WooCommerce Bulk Edit в меню Товары или в верхней панели Bulk Edit Products
-По умолчанию фильтры не применяются, и при нажатии кнопки Загрузить продукты все доступные продукты будут загружены в редактор.
-Вы можете легко фильтровать товары по названию, обычной цене и цене продажи, категориям, атрибутам, артикулу и тегам:
-Вы можете редактировать каждый продукт, просто щелкнув по его полям в таблице. Введите новое значение и просто нажмите "Ввод" или щелкните на другой ячейке.
-После завершения редактирования поля в его верхнем левом углу появится маленький красный треугольник, указывающий на то, что это значение было изменено и будет обновлено в базе данных.
---Примечание! Изменения сохраняются с помощью кнопки Сохранить изменения или при переходе на другую страницу продуктов!
-Чтобы использовать функциональность массового редактирования, вы должны сначала выбрать строки.
-Нажмите Редактировать кнопку, чтобы отобразить диалоговое окно Edit. Выберите необходимые параметры и введите новые значения.
-Нажмите OK, чтобы подтвердить изменения.
---Примечание! Для массового редактирования изображений сначала выберите продукты, а затем щелкните по одному из выбранных.
---Внимание! Если вы удаляете варианты, используя опцию "Переместить в корзину" по умолчанию, а родительский вариант все еще существует, вариант не будет удален. Удалять следует только варианты с выбранным параметром "Удалить навсегда".
	Доп информация:
По умолчанию в электронной таблице отображается только часть доступных полей. Чтобы просмотреть другие поля, нажмите кнопку Показать / Скрыть поля внизу и выберите, какие столбцы отображать.
